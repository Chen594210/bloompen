# BloomPen（妙笔生花）

AI-powered WeChat Mini Program for freelancers and small businesses to generate multi-platform marketing content.

## Features
- AI brand slogan generation
- Social media post generation
- Multi-platform copywriting
- One-click rewriting

## Tech Stack
- Spring Boot
- PostgreSQL
- React
- Taro
- Docker
- LLM API
